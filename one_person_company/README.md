# 🏢 One Person Company - AI Agent 驱动的一人公司系统

> 让 AI 帮你赚钱的全自动化商业系统

## 🎯 愿景

打造一个由 7 位 AI 专家组成的虚拟公司，实现：
- **电商业务**：自动选品、采购、销售（抖音/视频号/TikTok）
- **金融投资**：全球股市智能交易
- **自动运转**：最小化人工干预，最大化收益

## 🚀 快速开始

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 配置 API 密钥
cp config/api_keys.yaml.example config/api_keys.yaml
# 编辑填入你的 API 密钥

# 3. 启动系统
python main.py

# 4. 查看仪表盘
open http://localhost:8080
```

## 📈 核心指标

- **GMV**：电商总成交额
- **ROI**：投资回报率
- **利润率**：净利润/总收入
- **自动化率**：无人干预完成率
- **决策准确率**：正确决策占比
