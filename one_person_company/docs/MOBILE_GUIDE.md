## 🚀 方案一：企业微信机器人（最快上手）

### 优点
- ✅ 5分钟配置完成
- ✅ 实时推送到手机
- ✅ 支持Markdown美化
- ✅ 无需额外App

### 配置步骤

#### 1. 创建企业微信群机器人
1. 打开企业微信App（如果没有，下载并用微信登录）
2. 创建一个群（或使用现有群）
3. 点击群右上角 `...` → `添加群机器人`
4. 点击 `新创建一个机器人`
5. 输入名称：`一人公司助手`
6. 创建成功后，复制 **Webhook地址**

#### 2. 配置系统
编辑 `config/api_keys.yaml`：
```yaml
notification:
  wechat_bot:
    enabled: true
    webhook_url: "你复制的Webhook地址"
```

#### 3. 测试推送

## 🤖 方案二：Telegram Bot（支持双向交互）

### 优点
- ✅ 支持发送命令交互
- ✅ 全球可用，不受限制
- ✅ 开放API，扩展性强

### 配置步骤

#### 1. 创建Bot
1. 在Telegram搜索 `@BotFather`
2. 发送 `/newbot`
3. 输入Bot名称：`OnePersonCompany`
4. 输入Bot用户名：`your_company_bot`（需唯一）
5. 获得 **Bot Token**

#### 2. 获取Chat ID
1. 给你的Bot发一条消息
2. 访问：`https://api.telegram.org/bot<YOUR_TOKEN>/getUpdates`
3. 找到 `"chat":{"id": 123456789}` 中的数字

#### 3. 配置系统
```yaml
notification:
  telegram:
    enabled: true
    token: "你的Bot Token"
    chat_id: "你的Chat ID"
```
