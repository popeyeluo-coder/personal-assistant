# 证券分析专家

## 概述

本skill提供全面的证券分析能力，整合资金、技术、公司基本盘、行业趋势、宏观环境五大维度的分析框架。通过结合传统金融分析理论与现代数据科学技术，为投资者提供系统化的证券投资决策支持。

## 核心能力

本skill提供五个相互关联的分析维度：

### 1. 资金面分析
- 主力资金流向追踪
- 大单交易分析
- 北向资金与南向资金监控
- 机构持仓变化
- 融资融券数据分析

### 2. 技术面分析
- K线形态识别与解读
- 趋势线与支撑/阻力位分析
- 技术指标计算与解读：
  - MA（移动平均线）
  - MACD（指数平滑异同移动平均线）
  - RSI（相对强弱指标）
  - KDJ（随机指标）
  - 布林带（BOLL）
- 成交量分析
- 量价关系研判

### 3. 公司基本盘分析
- 财务报表三表分析：
  - 资产负债表：资产结构、债务状况
  - 利润表：收入、费用、盈利能力
  - 现金流量表：现金流产生与运用
- 关键财务指标：
  - ROE（净资产收益率）
  - ROA（总资产收益率）
  - 毛利率、净利率
  - 资产负债率
  - 流动比率、速动比率
  - EPS（每股收益）
  - PE、PB、PS等估值指标
- 公司治理结构
- 管理层与股东结构
- 行业地位与竞争优势

### 4. 行业趋势分析
- 行业生命周期判断（导入期/成长期/成熟期/衰退期）
- 行业竞争格局分析（波特五力模型）
- 市场规模与增长潜力
- 行业政策环境
- 技术变革与创新趋势
- 行业估值水平对比
- 产业链上下游分析

### 5. 宏观环境分析
- GDP与经济增长趋势
- 货币政策（利率、存款准备金率）
- 财政政策（税收、政府支出）
- 通货膨胀与CPI/PPI
- 国际贸易环境
- 汇率变动影响
- 金融市场整体流动性
- 系统性风险评估

### 6. 结构化输出能力（新增）
针对每只股票提供深度分析：

#### 6.1 涨跌原因分析
- **为什么上涨**：分析上涨驱动因素
  - 公司利好消息（业绩预增、订单大增、新产品发布等）
  - 行业利好政策（政策支持、行业标准等）
  - 宏观经济因素（利率下行、流动性宽松等）
  - 资金面因素（主力资金流入、外资加仓等）
  - 技术面因素（突破阻力位、技术指标金叉等）
  
- **为什么下跌**：分析下跌驱动因素
  - 公司利空消息（业绩下滑、监管处罚、诉讼等）
  - 行业利空政策（政策收紧、行业整顿等）
  - 宏观经济风险（利率上行、流动性收紧等）
  - 资金面因素（主力资金流出、外资减仓等）
  - 技术面因素（跌破支撑位、技术指标死叉等）

#### 6.2 趋势影响分析
- **国际形势影响**：
  - 国际贸易关系变化（中美贸易、一带一路等）
  - 国际金融市场波动（美股、欧股、港股等）
  - 大宗商品价格波动（原油、黄金、铜等）
  - 地缘政治风险（战争、制裁等）
  - 全球流动性环境（美联储加息/降息等）

- **宏观趋势影响**：
  - 经济周期阶段（复苏/繁荣/衰退/萧条）
  - 货币政策走向（宽松/中性/紧缩）
  - 财政政策力度（积极/稳健/收紧）
  - 通胀水平与预期（CPI、PPI走势）
  - 汇率变动对进出口企业影响

- **微观趋势影响**：
  - 行业景气度变化（供需关系、库存水平等）
  - 公司基本面变化（业绩、订单、产能等）
  - 市场情绪变化（投资者信心、风险偏好等）
  - 竞争格局变化（竞争对手动态、市场份额等）

#### 6.3 看盘要点分析
- **需要关注的关键指标**：
  - 价格：开盘价、收盘价、最高价、最低价
  - 成交量：量能变化、量价关系
  - 资金流向：主力资金、北向资金、机构动向
  - 技术指标：MA、MACD、RSI、KDJ、BOLL
  - 市场情绪：涨跌停数量、涨停打开率、炸板率

- **为什么关注这些指标**：
  - 价格反映市场供需关系
  - 成交量反映资金活跃度和趋势强度
  - 资金流向反映主力资金态度
  - 技术指标反映趋势和买卖信号
  - 市场情绪反映投资者心理

#### 6.4 未来走势预测与操作建议
- **明天走势预测**：
  - 趋势方向：上涨/下跌/震荡
  - 波动范围：预测最高价、最低价
  - 成交量预测：放量/缩量/平稳

- **为什么会这样**：
  - 技术面分析：趋势、形态、指标信号
  - 资金面分析：主力资金动向
  - 基本面分析：公司基本面变化
  - 宏观面分析：宏观环境影响
  - 市场情绪：投资者心理变化

- **该怎么操作**：
  - 买入：条件、价格区间、仓位控制
  - 持有：理由、风险点、关注指标
  - 卖出：条件、价格区间、止盈止损
  - 观望：原因、等待信号、入场时机

- **未来利好兑现后影响**：
  - 短期影响：股价涨跌空间、波动加剧
  - 中期影响：基本面改善、估值修复
  - 长期影响：竞争优势、成长空间
  - 风险提示：利好兑现后的回调风险

### 7. A股大盘分析（新增）
#### 7.1 大盘整体分析
- **指数走势**：上证指数、深证成指、创业板指、科创50等
- **市场情绪**：涨跌停数量、涨跌比、成交量
- **资金流向**：北向资金、南向资金、两融余额

#### 7.2 行业板块分析

- **弱势板块**：领跌板块、跌幅、资金流出
- **板块轮动**：资金在不同板块间流动
- **行业景气度**：各行业当前景气度水平

#### 7.3 备选投资标的推荐
- **推荐关注的行业**：
  - 为什么关注该行业
  - 行业基本面分析
  - 政策支持情况
  - 未来增长潜力
  - 风险因素

- **推荐关注的股票**：
  - 股票代码和名称
  - 推荐理由
  - 基本面亮点
  - 技术面信号
  - 风险提示
  - 入场时机

## 分析工作流

### 第一阶段：数据获取
1. 确定分析标的（股票代码/基金代码）
2. 使用数据源获取：
   - 历史交易数据（价格、成交量）
   - 财务数据（财报数据）
   - 资金流向数据
   - 行业与宏观数据
   - 新闻资讯与研报

### 第二阶段：五维分析
按照以下顺序执行五大维度分析：

**步骤1：宏观环境分析**
- 分析当前宏观经济形势
- 评估货币政策与财政政策影响
- 判断系统性风险水平

**步骤2：行业趋势分析**
- 确定行业所处生命周期阶段
- 分析行业竞争格局
- 评估行业增长前景

**步骤3：公司基本盘分析**
- 解读财务报表三表
- 计算关键财务指标
- 评估公司竞争力与护城河
- 判断公司估值水平

**步骤4：技术面分析**
- 绘制K线图与走势图
- 计算关键技术指标
- 识别趋势与形态
- 判断支撑/阻力位

**步骤5：资金面分析**
- 追踪主力资金动向
- 分析大单交易
- 评估机构持仓变化
- 判断市场情绪

### 第三阶段：综合研判
1. 综合五大维度分析结果
2. 识别各维度之间的共振或矛盾
3. 形成投资建议（买入/持有/卖出/观望）
4. 给出风险提示与关键关注点
5. 设定止损位与目标位（如适用）

## 分析原则与注意事项

### 核心原则
1. **多维综合**：不依赖单一维度，综合五大维度分析
2. **数据驱动**：基于客观数据，避免主观臆断
3. **风险优先**：始终将风险控制放在首位
4. **动态调整**：市场变化时及时调整判断
5. **长期视角**：区分短期波动与长期趋势

### 风险提示
- 历史表现不代表未来收益
- 市场存在系统性风险与非系统性风险
- 分析结果仅供参考，不构成投资建议
- 投资有风险，入市需谨慎

### 局限性说明
- 数据来源可能存在延迟或错误
- 分析模型可能无法预测突发事件
- 市场情绪和投资者心理难以量化
