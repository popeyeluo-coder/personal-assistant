## 分析维度详解

### 1. 资金面分析
查看资金流向、大单交易、机构持仓等
- 主力资金追踪
- 北向资金监控
- 融资融券数据
- 市场情绪判断

### 2. 技术面分析
基于价格和成交量的技术指标分析
- K线形态识别
- 趋势分析
- 技术指标（MA、MACD、RSI、KDJ、BOLL）
- 支撑阻力位

### 3. 基本面分析
基于财务报表和公司价值分析
- 财务三表分析
- 关键指标计算
- 竞争优势评估
- 估值水平判断

### 4. 行业趋势分析
分析行业生命周期和竞争格局
- 行业生命周期
- 竞争格局
- 增长前景
- 政策环境

### 5. 宏观环境分析
分析宏观经济和政策环境
- GDP与经济增长
- 货币政策
- 财政政策
- 通胀与汇率

## 最佳实践

### 1. 多维综合
不要只看单一维度，建议：
- 先宏观，后行业，再公司
- 基本面为主，技术面为辅
- 资金面验证

### 2. 风险控制
始终将风险放在首位：
- 设置止损位
- 分散投资
- 不满仓单一股票

### 3. 长期视角
- 区分短期波动和长期趋势
- 关注公司长期价值
- 不要频繁交易

### 4. 持续学习
- 定期更新行业知识
- 学习新的分析方法
- 关注政策变化

## 常见问题

### Q1：如何选择股票？
A：建议自上而下的方法：
1. 先看宏观环境，判断市场趋势
2. 再看行业趋势，选择景气度高的行业
3. 最后看公司基本面，选择优质公司

### Q2：什么时候买入？
A：综合考虑以下因素：
- 基本面：估值合理或低估
- 技术面：趋势向上或企稳
- 资金面：主力资金流入
- 行业：景气度提升

### Q3：什么时候卖出？
A：出现以下信号考虑卖出：
- 基本面：估值过高、基本面恶化
- 技术面：趋势破位、技术指标死叉
- 资金面：主力资金大幅流出
- 行业：景气度下降

### Q4：如何控制风险？
A：
- 分散投资，不要集中持股
- 设置止损位，严格执行
- 控制仓位，不要满仓
- 不要追涨杀跌
