# 基本面分析参考手册

## 财务报表分析

### 1. 资产负债表（Balance Sheet）

**核心公式**：
```
资产 = 负债 + 所有者权益
```

**重要科目分析**：

#### 流动资产
- **货币资金**：现金及银行存款，流动性最强
- **应收账款**：客户欠款，关注坏账风险
- **存货**：原材料、在产品、库存商品，关注周转率
- **预付款项**：预先支付的款项

#### 非流动资产
- **固定资产**：房屋、设备等长期资产
- **无形资产**：专利、商标等
- **长期股权投资**：对其他公司的投资

#### 流动负债
- **短期借款**：一年内到期的借款
- **应付账款**：欠供应商款项
- **预收款项**：预先收取的款项

#### 非流动负债
- **长期借款**：一年以上的借款
- **应付债券**：发行的债券

#### 所有者权益
- **实收资本**：股东投入的资本
- **资本公积**：超过面值的溢价等
- **盈余公积**：从净利润中提取的公积金
- **未分配利润**：累计未分配的利润

**分析要点**：
- 资产结构是否合理
- 负债水平是否过高
- 权益是否稳定增长

### 2. 利润表（Income Statement）

**核心公式**：
```
营业收入 - 营业成本 = 营业利润
营业利润 + 营业外收支 = 利润总额
利润总额 - 所得税 = 净利润
```

**关键指标**：

#### 盈利能力指标
- **毛利率** = (营业收入 - 营业成本) / 营业收入
- **净利率** = 净利润 / 营业收入
- **营业利润率** = 营业利润 / 营业收入
- **扣非净利润**：扣除非经常性损益后的净利润

#### 增长能力指标
- **营收增长率** = (本期营收 - 上期营收) / 上期营收
- **净利润增长率** = (本期净利润 - 上期净利润) / 上期净利润

**分析要点**：
- 收入质量（是否持续、稳定）
- 成本控制能力
- 利润质量（是否来自主业）

### 3. 现金流量表（Cash Flow Statement）

**三大活动**：

#### 经营活动现金流
- 企业核心业务产生的现金流
- 健康的企业：经营现金流持续为正且增长
- 关注：经营活动现金流净额

#### 投资活动现金流
- 购建固定资产、对外投资等
- 快速扩张期通常为负
- 关注：投资回报情况

#### 筹资活动现金流
- 借款、分红、股权融资等
- 健康期：适当分红、适度借款

**分析要点**：
- 自由现金流 = 经营现金流 - 资本支出
- 现金流是否匹配净利润
- 资金链是否安全

## 关键财务指标

### 1. 盈利能力指标

#### ROE（Return on Equity - 净资产收益率）
```
ROE = 净利润 / 平均净资产
```

**解读**：
- 衡量股东投入资本的回报率
- 巴菲特最看重的指标
- 好公司：持续15%以上
- 优秀公司：持续20%以上

#### ROA（Return on Assets - 总资产收益率）
```
ROA = 净利润 / 平均总资产
```

**解读**：
- 衡量资产使用效率
- 一般要求：5%以上
- 高ROA说明资产运用效率高

#### ROIC（Return on Invested Capital - 投入资本回报率）
```
ROIC = 税后经营利润 / (股东权益 + 有息负债 - 现金)
```

**解读**：
- 衡量投入资本的回报
- 考虑了资本成本
- ROIC > WACC为正收益

### 2. 估值指标

#### PE（市盈率）
```
PE = 股价 / 每股收益
```

**解读**：
- 回本年数
- 行业平均：15-25
- 低PE可能被低估，也可能成长性差
- 高PE可能被高估，也可能高成长

#### PB（市净率）
```
PB = 股价 / 每股净资产
```

**解读**：
- 净资产倍数
- 一般<2较合理
- <1可能被低估（需确认资产质量）
- 适用于资产密集型行业

#### PS（市销率）
```
PS = 股价 / 每股收入
```

**解读**：
- 收入倍数
- 适用于亏损公司或高成长公司
- 行业差异大

#### PEG（市盈率相对盈利增长比率）
```
PEG = PE / 净利润增长率
```

**解读**：
- PE = 1合理
- PEG < 1低估
- PEG > 1高估

### 3. 财务健康指标

#### 资产负债率
```
资产负债率 = 总负债 / 总资产
```

**解读**：
- 杠杆水平
- 一般<60%较安全
- >70%风险较高
- 不同行业标准不同

#### 流动比率
```
流动比率 = 流动资产 / 流动负债
```

**解读**：
- 短期偿债能力
- 标准：2左右
- <1短期偿债压力大
- 过高可能资金效率低

#### 速动比率
```
速动比率 = (流动资产 - 存货) / 流动负债
```

**解读**：
- 更严格的短期偿债能力
- 标准：1左右
- <1短期偿债风险大

### 4. 运营效率指标

#### 存货周转率
```
存货周转率 = 营业成本 / 平均存货
存货周转天数 = 365 / 存货周转率
```

**解读**：
- 存货周转效率
- 越高越好
- 存货积压是风险信号

#### 应收账款周转率
```
应收账款周转率 = 营业收入 / 平均应收账款
应收账款周转天数 = 365 / 应收账款周转率
```

**解读**：
- 应收账款回收效率
- 越高越好
- 需关注坏账风险

#### 总资产周转率
```
总资产周转率 = 营业收入 / 平均总资产
```

**解读**：
- 资产使用效率
- 越高越好
- 行业差异大

## 杜邦分析法

### 杜邦公式
```
ROE = 销售净利率 × 总资产周转率 × 权益乘数
```

### 分解分析
```
ROE = (净利润 / 营业收入) × (营业收入 / 总资产) × (总资产 / 净资产)
```

**分析框架**：

#### 1. 销售净利率（盈利能力）
- 产品盈利能力强
- 成本控制好
- 产品有溢价能力

#### 2. 总资产周转率（运营效率）
- 资产使用效率高
- 周转快
- 资产质量好

#### 3. 权益乘数（财务杠杆）
- 负债水平高
- 放大收益但也放大风险
- 需评估负债质量

**优秀公司的杜邦分析特征**：
- 高净利率：品牌、技术优势
- 高周转率：运营效率高
- 适度杠杆：合理利用财务杠杆

## 财务造假识别

### 常见造假手段

#### 1. 收入造假
- 虚构收入
- 提前确认收入
- 关联交易虚增收入

#### 2. 成本造假
- 少计成本
- 费用资本化
- 隐瞒费用

#### 3. 资产造假
- 虚增资产
- 资产评估虚高
- 关联交易输送利益

### 识别信号

#### 红色警报
- 现金流与净利润严重不符
- 应收账款异常增长
- 存货异常增长
- 毛利率异常高于同行
- 审计意见非标准
- 频繁更换审计机构
- 高管频繁变动
- 关联交易复杂

#### 验证方法
- 对比同行数据
- 分析现金流
- 检查会计政策
- 关注审计意见
- 查阅监管处罚记录

## 行业分析方法

### 波特五力模型

#### 1. 现有竞争者竞争程度
- 竞争者数量和规模
- 行业增长速度
- 固定成本高低
- 退出壁垒高低

#### 2. 新进入者威胁
- 进入壁垒高低
- 政策准入门槛
- 规模经济要求
- 品牌和渠道要求

#### 3. 替代品威胁
- 替代品可获得性
- 替代品价格
- 用户转换成本

#### 4. 供应商议价能力
- 供应商集中度
- 替代供应商可获得性
- 前向整合能力

#### 5. 买方议价能力
- 买方集中度
- 产品差异化程度
- 买方信息透明度

### 行业生命周期

#### 导入期
- 市场增长率低
- 技术不成熟
- 竞争者少
- 投资风险高

#### 成长期
- 市场增长率高
- 技术逐步成熟
- 竞争者增加
- 投资机会多

#### 成熟期
- 市场增长率放缓
- 技术成熟稳定
- 竞争格局稳定
- 现金流稳定

#### 衰退期
- 市场增长率低或负
- 技术落后
- 竞争者退出
- 利润下滑

### 行业估值

#### 相对估值
- 市盈率对比
- 市净率对比
- 市销率对比
- EV/EBITDA对比

#### 绝对估值
- DCF（现金流折现）
- DDM（股息折现）
- 相对PE/PS估值

## 基本面分析注意事项

### 1. 质量重于数量
- 财务数据质量
- 会计政策连续性
- 信息披露透明度

### 2. 历史不等于未来
- 关注未来增长
- 行业发展趋势
- 公司战略规划

### 3. 行业对比
- 同行业横向对比
- 识别竞争优势
- 关注行业集中度

### 4. 长期视角
- 短期波动正常
- 关注长期趋势
- 坚持价值投资

### 5. 风险意识
- 财务风险
- 经营风险
- 行业风险
- 系统性风险
