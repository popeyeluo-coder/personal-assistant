# 宏观环境与行业分析参考手册

## 宏观环境分析

### 1. 经济周期分析

#### 经济周期阶段
```
扩张期 → 峰值 → 收缩期 → 谷底 → 扩张期
```

#### 各阶段特征

**扩张期**
- GDP增长加速
- 就业率上升
- 消费和投资增加
- 企业盈利改善
- 央行可能加息

**峰值期**
- 经济过热迹象
- 通胀压力上升
- 加息预期增强
- 资产价格高位

**收缩期**
- GDP增长放缓或负增长
- 失业率上升
- 消费和投资下降
- 企业盈利恶化
- 央行可能降息

**谷底期**
- 经济触底
- 政策宽松预期
- 筑底过程
- 投资机会显现

### 2. 货币政策分析

#### 主要指标

#### 利率
- **存款准备金率**：银行需缴存的准备金比例
  - 上调：紧缩流动性
  - 下调：释放流动性

- **政策利率**（LPR、MLF利率）：
  - 上调：收紧货币
  - 下调：放松货币

- **市场利率**（国债收益率、Shibor）：
  - 反映市场资金成本
  - 影响估值水平

#### 货币供应量
- **M0**：流通中的现金
- **M1**：M0 + 活期存款（狭义货币）
- **M2**：M1 + 定期存款（广义货币）

**解读**：
- M2增速 > GDP增速 + 通胀率：流动性宽松
- M2增速 < GDP增速 + 通胀率：流动性紧缩
- M1-M2剪刀差：反映企业资金活跃度

#### 信贷数据
- **社融规模**：实体经济融资总量
- **新增人民币贷款**：银行放款情况
- **企业中长期贷款**：企业投资意愿

**解读**：
- 社融超预期：经济活力强
- 中长期贷款占比高：企业投资意愿强

### 3. 财政政策分析

#### 财政收入
- 税收收入
- 非税收入

#### 财政支出
- 一般性支出
- 投资性支出
- 民生支出

#### 关键指标
- **财政赤字率**：赤字 / GDP
  - 3%：国际警戒线
  - 超过3%：积极财政政策

- **政府债务率**：政府债务 / GDP
  - 60%：国际警戒线

**解读**：
- 积极财政政策：减税、增加支出
- 紧缩财政政策：加税、减少支出

### 4. 通胀分析

#### CPI（消费者物价指数）
- 衡量居民消费品价格变化
- 同比变化率：通胀率
- 核心CPI：剔除食品和能源

#### PPI（生产者物价指数）
- 衡量工业品出厂价格变化
- 反映上游成本压力

**通胀水平分类**：
- < 2%：通缩风险
- 2-3%：温和通胀
- 3-5%：通胀偏高
- > 5%：高通胀

**对市场影响**：
- 温和通胀：利好股市
- 高通胀：央行可能加息，利空股市
- 通缩：经济衰退风险

### 5. 汇率分析

#### 人民币汇率
- **兑美元汇率**：
  - 升值：进口利好、出口利空
  - 贬值：进口利空、出口利好

- **汇率指数**：对一篮子货币

**对股市影响**：
- 人民币升值：利好进口、航空等
- 人民币贬值：利好出口、纺织等

## 行业分析框架

### 1. 行业分类

#### 按生命周期
- **成长型行业**：科技、新能源、人工智能
- **成熟型行业**：银行、地产、家电
- **衰退型行业**：传统制造业（部分）

#### 按经济周期敏感度
- **周期性行业**：有色、钢铁、航运
- **防御性行业**：食品饮料、医药、公用事业
- **成长性行业**：科技、传媒、电子

#### 按产业链位置
- **上游**：资源、原材料（石油、煤炭、有色金属）
- **中游**：制造、加工（机械、化工、汽车）
- **下游**：消费、服务（零售、金融、医疗）

### 2. 行业竞争格局

#### 完全竞争
- 大量小企业
- 产品同质化
- 无定价权
- 利润率低
- 例：农业、纺织

#### 垄断竞争
- 较多企业
- 产品有差异化
- 部分定价权
- 例：消费、医药

#### 寡头垄断
- 少数大企业
- 高进入壁垒
- 强定价权
- 例：银行、电信、航空

#### 完全垄断
- 唯一企业
- 政府管制
- 例：公用事业

### 3. 行业驱动因素

#### 政策驱动
- **政策利好**：新能源、半导体、军工
- **政策压制**：房地产、教育、金融（有时）

#### 技术驱动
- 技术革命带来新需求
- 例：人工智能、5G、新能源汽车

#### 消费升级
- 收入增长带来消费升级
- 例：高端白酒、奢侈品、医疗服务

#### 人口结构
- 老龄化：医疗、养老
- 年轻化：消费、娱乐

#### 成本变化
- 原材料成本：影响利润率
- 人力成本：影响竞争力

### 4. 行业估值

#### 绝对估值
- DCF模型
- DD模型

#### 相对估值
- 行业PE
- 行业PB
- 行业PS
- 行业EV/EBITDA

#### 历史对比
- 当前估值 vs 历史区间
- 高估/低估判断

#### 横向对比
- 本行业 vs 其他行业
- 本行业 vs 历史平均

### 5. 行业景气度

#### 高景气度特征
- 行业增长率高
- 企业盈利改善
- 产能利用率高
- 投资意愿强
- 政策支持

#### 低景气度特征
- 行业增长率低或负增长
- 企业盈利恶化
- 产能过剩
- 投资意愿弱
- 政策限制

#### 景气度跟踪指标
- 行业销量/产量
- 价格指数
- 库存水平
- 订单情况
- 投资数据

### 6. 重点行业分析

#### 科技行业
**特点**：
- 高成长性
- 高研发投入
- 高风险高回报
- 技术迭代快

**分析要点**：
- 技术壁垒
- 研发能力
- 市场空间
- 商业模式

#### 消费行业
**特点**：
- 需求稳定
- 品牌重要
- 渠道关键
- 分红稳定

**分析要点**：
- 品牌力
- 渠道力
- 产品力
- 消费升级

#### 金融行业
**特点**：
- 高杠杆
- 强监管
- 顺周期
- 风险敏感

**分析要点**：
- 资产质量
- 净息差
- 不良率
- 资本充足率

#### 医药行业
**特点**：
- 需求刚性
- 政策影响大
- 研发周期长
- 知识产权重要

**分析要点**：
- 研发管线
- 专利保护
- 集采影响
- 政策导向

#### 地产行业
**特点**：
- 高杠杆
- 周期性强
- 政策敏感
- 现金流重要

**分析要点**：
- 销售数据
- 库存水平
- 债务结构
- 政策变化

#### 新能源行业
**特点**：
- 高增长
- 高景气
- 高估值
- 技术变化快

**分析要点**：
- 技术路线
- 成本下降
- 政策支持
- 市场空间

### 7. 行业轮动

#### 轮动逻辑
- 经济周期：周期行业先涨后跌
- 通胀预期：资源股受益
- 利率变化：金融、地产受益
- 政策导向：受政策支持的行业受益

#### 轮动策略
- 配对交易：多强空弱
- 行业ETF：快速切换
- 龙头公司：优先配置

## 宏观与行业的传导机制

### 1. 货币政策传导
```
利率变化 → 资金成本 → 企业投资 → 行业景气 → 股价
```

### 2. 财政政策传导
```
财政支出 → 需求增加 → 行业订单 → 企业盈利 → 股价
```

### 3. 汇率传导
```
汇率变化 → 进出口 → 外向型行业 → 企业盈利 → 股价
```

### 4. 通胀传导
```
通胀 → 原材料价格 → 上游行业 → 下游行业 → 股价
```

## 分析方法论

### 1. 自上而下
```
宏观环境 → 行业选择 → 个股筛选 → 投资决策
```

### 2. 自下而上
```
个股挖掘 → 行业验证 → 宏观确认 → 投资决策
```

### 3. 主题投资
```
主题逻辑 → 行业映射 → 个股选择 → 投资决策
```

### 4. 量化筛选
```
量化指标 → 股票池 → 基本面验证 → 投资决策
```
