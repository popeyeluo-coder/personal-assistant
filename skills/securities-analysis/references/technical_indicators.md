# 技术指标参考手册

## 趋势类指标

### 1. 移动平均线 (MA - Moving Average)

**原理**：对一定时期内的价格求平均，消除短期波动

**类型**：
- SMA（简单移动平均线）：所有数据权重相同
- EMA（指数移动平均线）：近期数据权重更大

**常用周期**：
- MA5：短期趋势
- MA10：短期趋势
- MA20：中期趋势（常用）
- MA60：中期趋势
- MA120：长期趋势（半年线）
- MA250：长期趋势（年线）

**交易信号**：
- 金叉：短期均线上穿长期均线，买入信号
- 死叉：短期均线下穿长期均线，卖出信号
- 多头排列：短中长期均线从上到下排列，强势
- 空头排列：短中长期均线从下到上排列，弱势

**优点**：
- 简单易懂
- 有效识别趋势
- 适合各种时间周期

**缺点**：
- 具有滞后性
- 震荡市场中容易产生假信号

### 2. MACD（指数平滑异同移动平均线）

**计算公式**：
```
DIF = EMA(12) - EMA(26)
DEA = EMA(DIF, 9)
MACD = (DIF - DEA) × 2
```

**组成**：
- DIF（快线）：12日EMA减去26日EMA
- DEA（慢线）：DIF的9日EMA
- MACD柱状图：(DIF - DEA) × 2

**交易信号**：
- 金叉：DIF上穿DEA，买入信号
- 死叉：DIF下穿DEA，卖出信号
- 底背离：股价创新低，MACD未创新低，反弹信号
- 顶背离：股价创新高，MACD未创新高，回调信号
- 零轴上金叉：强势金叉，信号更强
- 零轴下死叉：弱势死叉，信号更弱

**最佳使用场景**：
- 趋势性行情效果较好
- 结合成交量验证信号强度

## 震荡类指标

### 3. RSI（相对强弱指标）

**计算公式**：
```
RSI = 100 - [100 / (1 + RS)]
RS = 平均上涨幅度 / 平均下跌幅度
```

**取值范围**：0-100

**关键位置**：
- 70以上：超买区，可能回调
- 30以下：超卖区，可能反弹
- 50：多空分界线

**交易信号**：
- RSI > 70：超买，考虑减仓
- RSI < 30：超卖，考虑建仓
- 底背离：价格创新低，RSI未创新低，买入信号
- 顶背离：价格创新高，RSI未创新高，卖出信号

**优缺点**：
- 优点：提前预警、适合震荡市
- 缺点：强趋势中容易钝化（超买/超卖持续）

### 4. KDJ（随机指标）

**计算公式**：
```
RSV = (收盘价 - N日最低价) / (N日最高价 - N日最低价) × 100
K = SMA(RSV, M1)
D = SMA(K, M2)
J = 3K - 2D
```

**常用参数**：N=9, M1=3, M2=3

**取值范围**：
- K、D：0-100
- J：可超出0-100范围

**关键位置**：
- 80以上：超买区
- 20以下：超卖区

**交易信号**：
- K线上穿D线（金叉）：买入信号
- K线下穿D线（死叉）：卖出信号
- J值 > 100：严重超买，可能回调
- J值 < 0：严重超卖，可能反弹

**特点**：
- 比RSI更敏感
- 适合短线交易

## 通道类指标

### 5. 布林带（BOLL - Bollinger Bands）

**计算公式**：
```
中轨 = N日移动平均线
上轨 = 中轨 + K × N日标准差
下轨 = 中轨 - K × N日标准差
```

**常用参数**：N=20, K=2

**含义**：
- 中轨：20日移动平均线，代表价格趋势
- 上轨：压力位
- 下轨：支撑位
- 带宽：波动率指标，带宽越大波动越大

**交易信号**：
- 价格触及下轨：超卖，反弹概率大
- 价格触及上轨：超买，回调概率大
- 带口收缩（收口）：波动率降低，可能突破
- 价格突破上轨：强势上涨
- 价格跌破下轨：弱势下跌

**特点**：
- 动态调整，适应市场波动
- 可判断趋势强弱和买卖时机

### 6. SAR（抛物线转向指标）

**原理**：通过跟踪价格趋势，自动设定止损位

**交易信号**：
- SAR在价格下方：上升趋势，持有
- SAR在价格上方：下降趋势，空仓或做空
- 价格跌破SAR：平多仓
- 价格突破SAR：平空仓

**特点**：
- 适合趋势跟踪
- 自动设置止损位

## 成交量类指标

### 7. VOL（成交量）

**含义**：一定时期内的成交数量

**关键要点**：
- 量在价先：量能变化先于价格变化
- 放量上涨：买盘积极，趋势可靠
- 缩量上涨：上涨动力不足
- 放量下跌：抛压沉重
- 缩量下跌：抛压减轻，可能反弹

### 8. OBV（能量潮）

**计算公式**：
```
若今日收盘价 > 昨日收盘价：
    OBV = 昨日OBV + 今日成交量
若今日收盘价 < 昨日收盘价：
    OBV = 昨日OBV - 今日成交量
若今日收盘价 = 昨日收盘价：
    OBV = 昨日OBV
```

**交易信号**：
- OBV上升：买方力量占优
- OBV下降：卖方力量占优
- 股价上涨、OBV下跌：背离，可能见顶
- 股价下跌、OBV上涨：背离，可能见底

## 指标组合使用建议

### 组合1：趋势确认
```
MA + MACD
- MA判断趋势方向
- MACD确认趋势强度
- 双重验证提高准确度
```

### 组合2：买卖点确认
```
KDJ + 成交量
- KDJ提供买卖信号
- 成交量验证信号有效性
- 缩量信号需谨慎
```

### 组合3：趋势跟踪
```
BOLL + MACD
- BOLL判断支撑阻力
- MACD判断趋势方向
- 结合使用寻找最佳进出点
```

### 组合4：背离预警
```
RSI + 价格走势
- 寻找顶底背离
- 提前预警反转
- 结合其他指标确认
```

## 指标使用注意事项

### 1. 参数选择
- 不同股票适合不同参数
- 短线用短参数，长线用长参数
- 需要反复测试优化

### 2. 信号验证
- 单一指标容易产生假信号
- 多指标组合验证更可靠
- 结合成交量、市场环境判断

### 3. 适用场景
- 趋势市：趋势指标效果更好
- 震荡市：震荡指标效果更好
- 暴涨暴跌市：指标可能失效

### 4. 滞后性
- 所有技术指标都有滞后性
- 提前预警 vs 确认信号需权衡
- 不要机械照搬

### 5. 市场环境
- 不同市场环境适用不同策略
- 牛市、熊市、震荡市策略不同
- 需要动态调整

## 常见误区

### 1. 过度依赖指标
- 技术指标只是辅助工具
- 不能替代基本面分析
- 不能替代风险管理

### 2. 指标过多
- 指标过多容易互相矛盾
- 3-5个核心指标足够
- 专注于几个精通的指标

### 3. 机械照搬
- 每只股票特性不同
- 市场环境在变化
- 需要灵活运用

### 4. 忽视趋势
- 逆势操作风险大
- 顺应趋势是基本原则
- 不要试图抄底摸顶
